# Hello World Website

Preprosta HTML stran z DevOps CI/CD procesom.

## Tehnologije
- HTML
- CSS
- GitHub Pages
- GitHub Actions

## Struktura repozitorija

- `index.html` – glavna HTML stran
- `assets/` – CSS, slike 
- `.github/workflows/ci-cd.yml` – CI/CD workflow (smoke test)
- `smoke-check.py` – preprosti smoke test za preverjanje strani

## Deployment
Push na `master` branch → avtomatska objava
